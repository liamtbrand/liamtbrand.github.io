# Introductory Definitions

| Term | Definition |
| ---- | ---------- |
| Database | A very large, integrated, and commonly used collection of data. |
| Database Management System (DBMS) | A *DBMS* is a software package designed to store and manage *databases*. |
| Database System (DB) | A *database system* is a *DBMS* together with a *database*. |
| Data Model | A collection of concepts for describing data. Example: The Relational Model of Data. |
| Schema | An abstract description of *databases* using the given *data model*. |

Levels of Abstraction: Many *views* (*external schemata*), single *logical schema*, and single *physical schema*.

| Term | Definition |
| ---- | ---------- |
| View (External Schema) | Describes how users see the data. |
| Logical Schema | Defines logical structure. |
| Physical Schema | Describes the files and indices used. |
| Data Definition Language (DDL) | Defines the *database* *schema*. |
| Data Manipulation Language (DML) | Update data in the *database* (insert, modify, delete). |
| Query Language (QL) | Access and retrieve data from the *database*. |
