07/04/2015
MagicWorlds version 07.04.15a is done. Almost everything has been rebuilt from the ground up. This latest version supports maps edited with Tiled.
New Features:
+ Tiled map support - This allows for maps to be build much quicker using tiled which exports to xml and csv formatting.
+ Players - Other players appear in game.
+ Server Version Check - This will make sure the client and server are compatible before connection is fully made.
+ Portals - Allow for players to get between maps.
+ Client sided maps.
+ Server sided user data - Users now have their location and inventory saved upon disconnection.
- Removed old mapping system.
Fixes:
* Other players now update their orientations correctly.
* Built in pre-rendering of maps. This fixes previous frame rate issues.

--Liam

03/12/2013
MagicWorlds version 03.12.13a is done.
New Features:
1. Added Object Layer - This allows for objects to be placed on the object layer.
2. Added Objects - These will appear on the editing panel.
3. Server Version Check - This will make sure the client and server are compatible before connection is fully made.
4. New Map Format - I updated the map format to include both the terrain and object layers. Old maps will be converted to new ones when loaded, but they will only save in the newest format.
Updates:
1. Fixed crash when clicking on edges around map. (This was caused by an access violation in the map array..)
2. Fixed bugs caused by threads not synchronizing properly. (Let me know if they persist)
3. Fixed bug that client places tile right as it joins the server.
--Liam
